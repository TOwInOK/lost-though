About User Post Search

Post

GET: /{post_id}/ - for example, /6557b9f2417e299f07b8096a

Send the id and Get the Post.

POST: /{post_id}/edit

Send {RequsetPost}.

POST: /create

Send {RequsetPost} and Get create {Post}.

POST: /page/all

Send nothing and Get all posts in bd

POST: /page/{number}

Send number of page
1 number = 10 posts
2 number = 20 posts
end etc...

DELETE: /{post_id}/delete

Decided not to create another Api request type.
Send id
Send {Auth}
Get HttpResponse (we get this type of message everywhere, only the json is always custom if it is implied).
